Arquivos:

- styles.css
- script.js
- assets/ebook-preview.webp

COMO CONFIGURAR O WHATSAPP
1. Abra o arquivo script.js.
2. Localize: const WHATSAPP_NUMBER = "";
3. Coloque seu número com DDI e DDD, usando apenas números.
   Exemplo: const WHATSAPP_NUMBER = "5511999999999";

Enquanto o campo estiver vazio, os botões abrem o WhatsApp com a mensagem
pronta e permitem selecionar o contato manualmente.

PUBLICAÇÃO
Envie a pasta completa ao servidor. O arquivo inicial é index.html.
A página não depende de bibliotecas externas.
